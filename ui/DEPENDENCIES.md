# Dependencies Required

To use Tailwind CSS and Lucide React icons in this project, you need to install the following dependencies:

## Required Dependencies

```bash
npm install -D tailwindcss postcss autoprefixer
npm install lucide-react
```

## Package Details

1. **tailwindcss** - Utility-first CSS framework
2. **postcss** - Tool for transforming CSS with JavaScript plugins
3. **autoprefixer** - PostCSS plugin to parse CSS and add vendor prefixes
4. **lucide-react** - Beautiful & consistent icon toolkit for React

## Installation Steps

1. Install the dependencies:
   ```bash
   cd ui
   npm install -D tailwindcss postcss autoprefixer
   npm install lucide-react
   ```

2. The `tailwind.config.js` file has already been created in the `ui` directory.

3. The Tailwind directives have been added to `ui/src/index.css`.

4. You're ready to go! The components are already using Tailwind classes and Lucide icons.

## Note

If you're using Create React App (which this project appears to be using), Tailwind CSS should work out of the box with the configuration provided. No additional build configuration changes are needed.
